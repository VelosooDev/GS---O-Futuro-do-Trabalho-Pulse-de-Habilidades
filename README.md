# GS---O-Futuro-do-Trabalho-Pulse-de-Habilidades
O Pulse de Habilidades é uma solução de IoT que otimiza a colaboração no trabalho. Usando um dispositivo em cada mesa, a plataforma identifica em tempo real quem está disponível e tem a competência certa para ajudar em uma tarefa. Conectamos de forma inteligente e proativa as pessoas certas, na hora certa, eliminando a perda de tempo.
